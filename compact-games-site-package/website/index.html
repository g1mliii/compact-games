Easy start

It works with the launchers people already use.

Compact Games scans the game folders you already have, then lets you decide what to compress and what to leave alone.

Safer by design

Warns you before risky choices

If a game may not be a good fit for compression, Compact Games warns you before you make changes.

Your choice

Easy to restore later

Compression is reversible, so you are not locked into one choice.

Built for real cleanup

Less disk-space triage, fewer uninstall decisions.

The goal is simple: help you keep more of your installed games available on the same machine without turning storage cleanup into a chore.

Quick answers
Performance Most supported games keep working normally after compression.

Some titles are more sensitive, which is why the app warns you before you make risky changes.

Undo You can restore a game later if you change your mind.

Compact Games is meant to help you keep options open, not force one permanent setup.

Updates The same installer is used for first install and later updates.

You do not need a separate updater package or a different download flow.
