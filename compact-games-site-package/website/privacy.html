Privacy Policy

Last updated: April 8, 2026

Overview

Compact Games is a desktop utility for Windows that compresses game installs using the Windows Overlay Filter (WOF) compression API. This policy explains what data the application and this website collect, how that data is used, and your choices.

Data the Desktop App Collects

Compact Games does not require an account, login, or registration. The app:

  • Does not collect personal information, analytics, or telemetry.
  • Does not transmit your game library, file paths, or compression results to any server.
  • Makes outbound network requests only for: checking for updates via latest.json on GitHub Releases, and optionally fetching cover art from SteamGridDB if you provide an API key in settings.
  • May refresh the community unsupported-games list from the latest GitHub release asset. This is a read-only download with no user data sent.

Data This Website Collects

This website is a static site hosted on GitHub Pages. It does not use cookies, tracking scripts, or analytics services. GitHub Pages may log standard server access information (IP address, user agent, timestamps) as part of their infrastructure. See GitHub Pages data collection for details.

The website fetches release data from the GitHub API to display the latest version and download links. This request is made from your browser directly to GitHub's API servers.

Third-Party Services

  • GitHub — Hosts the source code, releases, and this website. Subject to GitHub's Privacy Statement.
  • SteamGridDB — Optional cover art provider. Only used if you manually configure an API key. Subject to SteamGridDB's own privacy policy.

Children's Privacy

Compact Games does not knowingly collect data from anyone, including children under the age of 13.

Changes to This Policy

If this policy changes, the update will be reflected on this page with a revised "Last updated" date.

Contact
