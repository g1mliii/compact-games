Frequently asked questions

What does Compact Games do?

It helps you save disk space by compressing supported game files on Windows.

Will this slow my games down?

Usually no, but some games are more sensitive than others. The app tries to warn you first.

Can I undo it later?

Yes. You can restore a game back to normal if you want.

Do I need a different installer to update?

No. The same installer is used for both first install and updates.

Is it Windows-only?

Yes. The current app is built for Windows 10 and Windows 11.

Can I download it from my phone or iPad?

Not directly. Compact Games installs with a Windows .exe, so use a Windows 10 or Windows 11 PC for the actual download and install.

Where can I report a problem?

You can use the contact form below or open a GitHub issue if you prefer.
