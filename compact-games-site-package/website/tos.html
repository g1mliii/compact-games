Description of Service

Compact Games is a free, open-source Windows desktop utility that applies Windows Overlay Filter (WOF) compression to game installs to reduce disk space usage. The Software is provided as a convenience tool and is not affiliated with any game publisher or platform.

Use at Your Own Risk

Compressing game files modifies how Windows stores them on disk. While the Software includes safety checks (DirectStorage detection, unsupported-games lists, running-process detection), no compression tool can guarantee compatibility with every game or system configuration.

  • You are responsible for verifying that your games work correctly after compression.
  • You are responsible for maintaining backups of critical data.
  • The decompression feature allows you to reverse compression, but this does not constitute a guarantee against data loss.
